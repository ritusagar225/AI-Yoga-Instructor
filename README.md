# 🧘 AI Yoga Pose Detection & Posture Correction System

An AI-powered real-time yoga posture detection and correction system built using **Computer Vision**, **Machine Learning**, and **Python**. The application detects human body landmarks, classifies yoga poses, analyzes posture quality, calculates joint angles, identifies mistakes, and provides intelligent corrective feedback in real time.

---

## 📌 Features

- 📷 Real-time webcam-based pose detection
- 🦴 Human skeleton tracking using MediaPipe Pose
- 🤖 Yoga pose classification
- 📐 Joint angle calculation
- 📊 Pose similarity scoring
- 🧠 Intelligent posture correction suggestions
- ⚖️ Stability analysis
- 🔄 Symmetry analysis
- 📈 Session analytics
- 🎯 Posture scoring system
- ⚡ Real-time performance (FPS monitoring)
- 🏗 Modular architecture for easy extension

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/AI-Yoga-Instructor.git

cd AI-Yoga-Instructor
```

Create virtual environment

```bash
python -m venv venv
```

Activate environment

Windows

```bash
venv\Scripts\activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python main.py
```
